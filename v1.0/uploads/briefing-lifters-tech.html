Parte 2 — Proposta de Redesign
06 — Conceito Criativo
Proposta de Redesign

Conceito: "Code in Motion"

Um site que respira tecnologia. Cada scroll revela conteúdo com animações que remetem a código sendo executado, dados fluindo e interfaces se construindo em tempo real. O visitante não lê — ele experimenta a expertise da Lifters. Background fixo com conteúdo que flui por cima, transições suaves, tipografia que se revela caractere por caractere, e contadores que pulsam com dados reais.

🎯 Diretrizes Criativas

AspectoDiretriz
TomTecnológico, premium, confiável. Sem ser genérico ou corporativo demais.
ExperiênciaSingle-page com scroll suave (ScrollSmoother). Background fixo, conteúdo desliza por cima.
AnimaçõesCada seção tem uma animação GSAP própria — nunca apenas fade-in. Combinação de SplitText, clip-path, scale+blur, parallax e counters.
Objetivo ComercialConverter visitantes em leads qualificados. Cada seção termina com um micro-CTA ou direcionamento para contato.
MobileResponsive-first. Animações simplificadas no mobile, mantendo impacto visual.
PerformanceGSAP via CDN, lazy-load de imagens, will-change estratégico, kill de triggers fora da viewport.

09 — Animações
Catálogo de Animações GSAP

Todas as animações usam GSAP v3 com plugins gratuitos (desde abril 2025, todos os plugins GSAP são 100% grátis).

1. ScrollSmoother + Background Fixo

Scroll suave com efeito de conteúdo fluindo sobre background travado.

JS // Estrutura HTML obrigatória // <div id="smooth-wrapper"><div id="smooth-content">...</div></div> gsap.registerPlugin(ScrollTrigger, ScrollSmoother); const smoother = ScrollSmoother.create({ wrapper: '#smooth-wrapper', content: '#smooth-content', smooth: 1.5, effects: true }); // Background fixo via CSS // .hero-bg { position: fixed; inset: 0; z-index: -1; }

2. SplitText — Reveal Caractere por Caractere

Texto aparece letra por letra com stagger. Ideal para hero e títulos.

JS gsap.registerPlugin(SplitText); const split = new SplitText('.hero-title', { type: 'chars, words' }); gsap.from(split.chars, { opacity: 0, y: 40, rotateX: -90, stagger: 0.03, duration: 0.8, ease: 'back.out(1.7)' });

3. Clip-Path Reveal

Conteúdo se revela com mask animada — efeito de "cortina" abrindo.

JS gsap.from('.reveal-block', { clipPath: 'inset(0 100% 0 0)', duration: 1.2, ease: 'power4.out', scrollTrigger: { trigger: '.reveal-block', start: 'top 80%' } });

4. Scale + Blur Reveal

Elemento surge do blur com scale crescente — profundidade e foco.

JS gsap.from('.card', { scale: 0.85, filter: 'blur(12px)', opacity: 0, duration: 1, ease: 'power3.out', stagger: 0.15, scrollTrigger: { trigger: '.cards-container', start: 'top 75%' } });

5. Counter Animado com Scrub

Números sobem de 0 ao valor final sincronizados com o scroll.

JS const counter = { val: 0 }; gsap.to(counter, { val: 10000, duration: 2, snap: { val: 1 }, scrollTrigger: { trigger: '.counter-section', start: 'top center', end: 'bottom center', scrub: 1 }, onUpdate: () => { el.textContent = counter.val.toLocaleString('pt-BR'); } });

6. SVG Line Draw (Timeline)

Linha SVG se desenha conforme scroll — perfeito para timeline de processo.

JS const path = document.querySelector('.timeline-line'); const length = path.getTotalLength(); gsap.set(path, { strokeDasharray: length, strokeDashoffset: length }); gsap.to(path, { strokeDashoffset: 0, ease: 'none', scrollTrigger: { trigger: '.timeline-section', start: 'top center', end: 'bottom center', scrub: true } });

7. Horizontal Slide-in com Stagger

Cards entram da direita um por um conforme o scroll avança.

JS gsap.from('.service-card', { xPercent: 100, opacity: 0, duration: 0.8, stagger: 0.2, ease: 'power3.out', scrollTrigger: { trigger: '.services-section', start: 'top 70%' } });

8. Parallax com data-speed

Elementos com velocidades diferentes criam profundidade 3D durante o scroll.

HTML + JS <!-- No HTML, adicionar atributo data-speed --> <div data-speed="0.8">...lento</div> <div data-speed="1.2">...rápido</div> // ScrollSmoother detecta automaticamente data-speed // quando effects: true está ativo. // Valores <1 = mais lento, >1 = mais rápido
10 — Técnico
Especificação Técnica

📦 CDN — Scripts GSAP (todos gratuitos)

HTML <!-- GSAP Core + Plugins (100% free desde abril 2025) --> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/gsap.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/ScrollTrigger.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/ScrollSmoother.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/SplitText.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/Flip.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/Observer.min.js"></script>

🏗️ Estrutura HTML Base

HTML <body> <!-- BG fixo --> <div class="hero-bg"> <!-- Grid pattern + particles --> </div> <!-- ScrollSmoother wrapper obrigatório --> <div id="smooth-wrapper"> <div id="smooth-content"> <section class="hero">...</section> <section class="about">...</section> <section class="services">...</section> <section class="verticals">...</section> <section class="cases">...</section> <section class="stack">...</section> <section class="process">...</section> <section class="contact">...</section> </div> </div> </body>

⚡ Performance

  • will-change: transform nos elementos animados
  • Kill ScrollTriggers fora da viewport
  • Lazy-load imagens com loading="lazy"
  • Debounce no resize para recalcular triggers
  • Preferir transform e opacity (GPU)
  • matchMedia para desativar animações pesadas no mobile

📱 Responsividade

  • ScrollSmoother.matchMedia para breakpoints
  • No mobile: desativar pin e parallax pesados
  • SplitText: usar words em vez de chars no mobile
  • Cards em coluna única abaixo de 768px
  • Animações mais curtas e simples no mobile
  • Touch-friendly: sem hover complexo
